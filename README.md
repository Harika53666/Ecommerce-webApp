## 🛍️ Harika's Cute Shop – Streamlit E-Commerce Web App 

## Project Overview :
Harika's Cute Shop is a simple and interactive e-commerce style web application built using Python and Streamlit. The application allows users to add products with images and prices and display them dynamically on a product listing page. It simulates the basic functionality of an online shopping platform where users can view available products and manage product listings through a user-friendly interface.

## Objective :
The main objective of this project is to understand how to build interactive web applications using Python. It focuses on implementing product display functionality, image uploading, and state management to create a mini e-commerce experience.

## Home Page :
The home page displays all available products that have been added to the application. Each product includes its image, name, and price. If no products are available, the application displays a message informing the user that there are currently no products in the store.

## Add Product Feature :
The application includes an "Add Product" page where users can add new items to the store. Users can enter the product name, specify the price, and upload an image for the product. Once the user clicks the add button, the product is stored and becomes visible on the home page.

## Image Upload Functionality :
The application allows users to upload product images in JPG or PNG format. The uploaded images are displayed along with the product details to create a more realistic online store layout.

## Session State Management :
The project uses Streamlit Session State to temporarily store product data. This ensures that the products added during the session remain visible while the application is running without needing a database.

## Technologies Used :
Python, Streamlit, Streamlit Session State, and basic web interface components.

## Learning Outcomes :
Through this project, practical experience was gained in building web applications using Streamlit, handling file uploads, managing application state, and creating dynamic user interfaces for displaying product information.
